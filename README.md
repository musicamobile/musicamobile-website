# Musicamobile Website

Website simples para a Musicamobile com logotipo e link de contato.

## Como adicionar o logo

1. Coloque o arquivo do logo na pasta `public/` com o nome `logo.png` (ou altere o nome no arquivo `app/page.tsx`)
2. Instale as dependências: `npm install`
3. Rode o projeto: `npm run dev`

## Deploy no Vercel

1. Conecte o repositório no Vercel
2. Configure o domínio musicamobile.com.br
3. Deploy automático
